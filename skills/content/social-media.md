# Social media content

## Overview
Write posts that fit each platform's voice, length, and format — then queue them. Nothing
is published without your approval.

## When to use
"Draft an Instagram post about…", "turn this into a LinkedIn version", "write a tweet/
thread on…", "make a TikTok hook", "caption for…", "5 posts for this offer".

## Workflow
1. Get the core idea/offer, the platform(s), and the goal (awareness / clicks / sales).
2. Write **native** to each platform:
   - **Instagram**: hook line → value → CTA → 3–8 relevant hashtags; emoji sparingly.
   - **X**: ≤280 chars, punchy; or a numbered thread with a strong opener.
   - **LinkedIn**: professional hook → short paragraphs → insight → soft CTA; few hashtags.
   - **TikTok / Shorts**: a spoken hook (first 2s) + on-screen text beats + caption.
   - **Threads / Facebook**: conversational; FB tolerates longer.
3. Offer platform variants from one idea (→ `content-strategy` for repurposing).
4. Save drafts; queue them. **Publishing goes through the approval gate** — Veyra shows the
   exact platform + content and asks before posting.

## Output format
- One block per platform, labeled, within the character limits.
- Hashtags grouped at the end where the platform expects them.

## Quality bar
Reads native to the platform (not generic); hook earns the scroll-stop; one clear CTA;
claims are honest; within limits.

## Examples
- "5 posts for a course launch." → IG, X, LinkedIn, Threads, TikTok variants, queued.
- "Turn this blog into a LinkedIn post." → hook + 3 insights + soft CTA.
