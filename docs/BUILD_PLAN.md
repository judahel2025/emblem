# Emblem 8-Hour Build Plan

## Goal

Build Emblem into a working local-first AI command center that already feels usable by the end of the first focused build window.

## Hour 1: Foundation

- Create the Emblem project folder.
- Start a local dashboard.
- Start a local backend.
- Add Ollama health and model detection.
- Add the first command interface.

Status: done.

## Hour 2: Local AI Brain

- Connect typed prompts to Ollama.
- Add mode routing:
  - Voice
  - Coding
  - Academic
  - Content
  - General
- Add model selector.

Status: first version done.

## Hour 3: Voice Command

- Add browser microphone input.
- Add speech-to-text through browser speech recognition for the prototype.
- Send voice transcript to the selected local model.
- Speak the response back through browser speech synthesis.
- Later upgrade:
  - offline Whisper transcription
  - offline Piper/Kokoro voice
  - wake phrase: "Hey Emblem"

Status: first version done.

## Hour 4: Custom IDE

- Add file explorer.
- Add code editor.
- Add generate/edit/explain/fix actions.
- Add safe file writing to project workspaces.
- Add run/test panel.

Status: next.

## Hour 5: Documents and Academic Studio

- Add document templates.
- Add academic essay mode.
- Add reports, proposals, study notes, and scripts.
- Export to Markdown first.
- Add DOCX/PDF export after.

Status: next.

## Hour 6: Research Layer

- Add online search.
- Summarize source results.
- Save research notes.
- Add citation helper.

Status: planned.

## Hour 7: Connectors

- Flutterwave:
  - API key settings
  - recent payments
  - successful/failed payment counts
  - webhook endpoint
- Gmail:
  - OAuth setup
  - inbox summary
  - draft replies
- Social:
  - local content queue
  - Buffer free option
  - Postiz/Mixpost self-hosted option

Status: planned.

## Hour 8: Polish and Demo Flow

- Add command history.
- Add approval queue.
- Add settings page.
- Improve empty/error states.
- Create a demo flow:
  - ask by voice
  - generate content
  - generate code
  - draft a document
  - show connector status

Status: planned.
