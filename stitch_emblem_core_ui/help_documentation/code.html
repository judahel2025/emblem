Frequently Asked Questions

How do I reset my organizational security key? expand_more

To reset your organizational security key, navigate to the Admin Console > Security Settings > Keys. From there, select 'Regenerate Master Key'. Please note that this will invalidate all current session tokens and requires 2FA confirmation from at least two organization owners.

What is the rate limit for the Enterprise API? expand_more

Standard enterprise tiers are allowed 10,000 requests per minute. High-frequency trading accounts or specialized intelligence nodes can request limit increases up to 1,000,000 RPM through their dedicated technical account manager.

Can I export my automation workflows to YAML? expand_more

Yes, EMBLEM supports full CI/CD portability. You can export any workflow by clicking 'Export' in the Automation Designer. The resulting file follows our standard Blueprint schema, compatible with most enterprise orchestration tools.

How are workspace credits calculated? expand_more

Credits are metered based on compute hours and storage volume. Data egress is free within your primary region. You can monitor your real-time consumption in the Billing Dashboard.
